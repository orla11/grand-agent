
# Grand Agent

# Overview

**Grand Agent** is a simple Python project using LangChain. The grand agent executes different multiple tasks by making use of different tools/agents.

# Getting Started

## Clone the Repository

`git clone https://github.com/orla11/grand-agent.git`

## Install Dependencies

`pipenv install`

## Set up API Keys

Ensure you set up your API keys in the .env file.

## Run the project

`make run-grand-agent`
